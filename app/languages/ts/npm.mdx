# NPM package panager

## What is NPM?

NPM (Node Package Manager) is a package manager for the JavaScript programming language.
It is the default package manager for the JavaScript runtime environment Node.js.
NPM allows developers to share and reuse code, manage dependencies, and automate tasks in their projects.

## How to update packages in a Next.js project

### Update all packages

To update packages in a Next.js project, you can use the following command.

```bash
npm update
```

### Update a specific package

To update a specific package, you can use the following command.

```bash
npm update <package-name>
```

### Check for outdated packages

To check for outdated packages, you can use the following command.

```bash
npm outdated
```

### Update NPM itself

To update NPM itself, you can use the following command.

```bash
npm install -g npm
```
